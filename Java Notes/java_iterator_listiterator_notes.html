★ Iterator Interface
Universal Cursor — works with List, Set & Queue. Provides a standardised way to read elements from any Collection object, one by one.
3 Methods
hasNext() → boolean. Is there a next element?
next() → E. Fetch & advance cursor
remove() → void. Safe removal of current element
How to Get Iterator
Iterator<String> itr = al.iterator(); // cursor starts BEFORE first element while(itr.hasNext()) { String city = itr.next(); System.out.println(city); }
★ Fail-Fast Behavior & ConcurrentModificationException
Fail-Fast Rule: ArrayList, HashSet etc. throw ConcurrentModificationException if the collection is structurally modified while iterating — unless you use itr.remove(). Enhanced for-loop internally uses Iterator, so same rule applies!
★ Index-Shift Bug — Normal For Loop Trap
3 Approaches to Fix the Index-Shift Problem
① Decrement i After Remove
al.remove(i); i--; // compensate shift
Step back after removal so next element isn't skipped.
② Traverse Backwards
for(int i=al.size()-1; i>=0; i--) al.remove(s); // no shift up
Removals only affect elements already visited.
③ Use Iterator (Best)
itr = al.iterator(); while(itr.hasNext()) if(match) itr.remove();
Safe, universal, works with all collections.
★ Limitations of Iterator
①Single direction only — cursor moves forward. Cannot go backward.
②Read & Remove only — cannot add or replace elements while iterating.
★ ListIterator Interface
Child of Iterator. The most powerful cursor — bidirectional, supports read, remove, add & replace. Limitation: works only with List objects (not Set or Queue).
All 9 Methods
Inherited from Iterator (3):
hasNext() next() remove()
New in ListIterator (6):
hasPrevious() previous() nextIndex() previousIndex() set(obj) add(obj)
📊 ListIterator Cursor Positions
Add & Replace While Iterating (Unique to ListIterator)
✦ Adding element after a match:
while(ltr.hasNext()) { String s = ltr.next(); if(s.equals("b")) ltr.add("X"); // inserts after "b" } // Result: [a, b, X, c, d, e, f]
✦ Replacing element on match:
while(ltr.hasNext()) { String s = ltr.next(); if(s.equals("b")) ltr.set("B"); // replaces "b" → "B" } // Result: [a, B, c, d, e, f]
